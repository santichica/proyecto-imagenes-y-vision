## Global Rules
- Keep the repository reproducible and easy to audit.
- Use config files for paths, seeds, and experiment parameters.
- Avoid hidden state in notebooks.
- Keep synthetic data metadata complete and machine-readable.
- Do not write outside the repository unless explicitly asked.
- Ask for confirmation before destructive or high-cost actions.
